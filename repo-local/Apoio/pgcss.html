  • OBS:

    • @import url(./reset.css); :Pode ser utilizado para importar um arquivo dentro pagina de estilo. Geralmente utilizado para que não tenha que repetir a estilização de toda pagina.

  • Propriedades

    • justify-content:

      • Determina como é feito a distribuição de forma horizontal.

    • text-decoration:

      • Decoração do texto Ex: underline

    • align-items:

      • Determina como é feito a distribuição de forma vertical

    • width:

      • Largura

    • height:

      • Altura

    • list-style:

      • Estilo da lista (li).

      • Atributos:

        • none: Limpa os pontos ou numeros da lista.

    • text-transform:

      • uppercase: Deixa o texto em caixa alta.

    • background-size:

      • cover: Distribui a img dentro do conteiner sem distorcer. Caso a img tenha um formato diferente do container ele cortara para caber.
