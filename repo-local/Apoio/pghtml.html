  • Propriedades comuns

    • class=

      • Utilizado para especificar o ID.

      • Pode ter ID repetido.

      • Para ser referenciado no CSS utiliza-se . .

    • id=

      • Utilizado para especificar o ID.

      • Não pode ter ID repetido. É usado para JS.

      • Para ser referenciado no CSS utiliza-se #.

  • Tags

    • <h1>, <h2>, <h3>, <h4>, <h5>

      • Utilizado para cabeçalhos.

      • h1 é utilizado apenas 1 vez no primeiro cabeçalho.

    • <p>

      • Utilizado para paragrafos

    • <div>

      • Utilizado para conteiner

      • Possui a caracteritica que por padrão ele vai do inicio ao fim da folha.

    • <span>

      • Utilizado para conteiner

      • Possui a caracteritica que por padrão ele não vai do inicio ao fim da folha, diferente da div.

    • <nav>

      • Utilizado para conteiner do menu de navegação

      • Semelhante a div porem com mais semantica

    • <form>

      • Utilizado para conteiner para formularios.

    • <ul>

      • Lista não ordenada<./p>

    • <ol>

      • Lista ordenada.

    • <li>

      • Item da lista.

    • <img>

      • Importar imagem.

      • A propriedade src="" define o endereço da imagem.

    • <a>

      • Cria um link.

      • A propriedade href="" define o endereço do link.

      • Pode ser utilizado # referenciando um ID dentro do href="" para navegar pela propria pagina.EX: <a href=".#ID">Apoio aprendizado</a></p></li>

      • A propriedade target="" define como a pagina endereço do link sera aberto.

        • target="_blank": Abre em outra pagina.

        • target="_parent":

        • target="_self": Abre na mesma pagina.

        • target="_top":

      • A propriedade title="" define o nome que ira aparecer quando passar o mose.

    • <button>

      • Cria botons.

      • A propriedade type="" define o tipo de botão.

    • <input>

      • Cria um input de informação.

      • A propriedade value="" define qual é de informação. Ex: value="nome"

      • A propriedade type="" define o tipo de informação.

        • type="text": Abre uma caixa de texto.

        • type="number": Abre uma caixa de texto, onde so pode ser acrecido numeros e sinais matematicos.

        • type="checkbos": Abre uma caixa de seleções multiplas.

        • type="radio": Abre uma caixa de seleções unicas.
